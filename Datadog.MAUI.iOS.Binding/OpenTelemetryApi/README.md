# Datadog.MAUI.iOS.OpenTelemetryApi

.NET bindings for the OpenTelemetry API framework (distributed with Datadog iOS SDK).

This package provides .NET MAUI bindings for OpenTelemetry tracing APIs used by the Datadog iOS SDK.

## Usage

This package is automatically included as a dependency when you use `Datadog.MAUI`. You typically don't need to reference it directly.
