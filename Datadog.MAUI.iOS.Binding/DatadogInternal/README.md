# Datadog.MAUI.iOS Binding

.NET bindings for Datadog iOS SDK module.

This package provides .NET MAUI bindings for the Datadog iOS SDK.

## Usage

This package is automatically included as a dependency when you use `Datadog.MAUI`. You typically don't need to reference it directly.
