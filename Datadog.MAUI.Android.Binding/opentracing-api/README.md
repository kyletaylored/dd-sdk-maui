# Datadog.MAUI.Android.OpenTracingApi

.NET bindings for the OpenTracing API (distributed with Datadog Android SDK).

This package provides .NET MAUI bindings for OpenTracing APIs used by the Datadog Android SDK.

## Usage

This package is automatically included as a dependency when you use `Datadog.MAUI`. You typically don't need to reference it directly.
