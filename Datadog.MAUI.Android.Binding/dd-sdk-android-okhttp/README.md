# Datadog.MAUI.Android Binding

.NET bindings for Datadog Android SDK module.

This package provides .NET MAUI bindings for the Datadog Android SDK.

## Usage

This package is automatically included as a dependency when you use `Datadog.MAUI`. You typically don't need to reference it directly.
